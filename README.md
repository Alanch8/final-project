# Final Project: Task App

![TaskApp](./public/taskapp-brand-logo.png)

## Project Setup

```sh
npm install
```

### Compile and Hot-Reload for Development

```sh
npm run dev
```

### Compile and Minify for Production

```sh
npm run build
```
